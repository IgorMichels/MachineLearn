Trabalho de Machine Learning¶

Igor Patrício Michels¶

Nessa segunda parte irei comparar a eficincia de quatro modelos de classificao quando testados sobre o dataset escolhido. So eles:

  • Regreso Logstica
  • Decision Tree
  • Random Forest
  • Gradient Boosting

Para tanto, declarei uma funo que testa alguns valores de parmetros e retorna o modelo que performa com os parmetros testados. Essa performance calculada por meio do cross-validation com $5$ grupos. Assim, para cada um dos modelos acima, foram propostos um conjunto de parmetros e os mesmos foram avaliados com o cross-validation, retornando o modelo dado com os melhores parmetros segundo a validao.

Alm dessa funo, tambm declarei duas funes para avaliar os resultados, uma retornando a matriz de confuso, bem como algumas outras mtricas, e outra que plota a curva ROC para o modelo.
